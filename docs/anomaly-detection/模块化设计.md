## 二、数据采集模块

### 2.1 采集范围

| 范围 | 数量 | 说明 |
|------|------|------|
| 沪深300 | ~300 | 核心，覆盖A股主要市值 |
| 中证500 | ~500 | 扩展，覆盖中盘 |
| 主要指数 | 5-6 | 沪深300、中证500、创业板等 |

### 2.2 采集策略

```
全量采集：每周一次（周末执行）
增量采集：每日收盘后（16:30后）

采集流程：
1. 获取股票列表（沪深300成分股）
2. 分批采集（每批50只，避免限速）
3. 写入数据库（UPSERT，去重）
```

### 2.3 采集数据源

| 数据源 | 用途 | 接口 |
|--------|------|------|
| AkShare | 日K线数据 | `ak.stock_zh_a_hist()` |
| AkShare | 指数成分 | `ak.index_zh_a_hist()` |
| AkShare | 股票信息 | `ak.stock_individual_info_em()` |

### 2.4 采集配置

```python
COLLECTOR_CONFIG = {
    "batch_size": 50,           # 每批采集数量
    "request_delay": 1.0,        # 请求间隔（秒）
    "retry_times": 3,            # 失败重试次数
    "lookback_years": 5,        # 回看5年数据
}
```

## 三、异常检测模块

### 3.1 检测指标体系

```
异常检测 = 价格异常 + 量能异常 + 波动异常 + 趋势异常
```

| 维度 | 指标 | 算法 | 阈值 |
|------|------|------|------|
| **价格** | 涨跌幅 | Z-Score | ±2.5σ |
| **量能** | 成交量 | IQR | Q3+1.5×IQR |
| **波动** | 振幅 | 百分位 | >95% 或 <5% |
| **趋势** | 均线破坏 | 交叉检测 | MA5/MA20 |

### 3.2 价格异常检测

**指标**：
- 涨跌幅（change_pct）
- 收盘价

**算法**：Z-Score（3σ原则）

```
判断逻辑：
1. 计算近30天涨跌幅的均值 μ 和标准差 σ
2. 当前涨跌幅 z = (当前值 - μ) / σ
3. |z| > 2.5 → 异常
```

**阈值**：
- |z| > 2.0：低风险
- |z| > 2.5：中风险
- |z| > 3.0：高风险（极端）

### 3.3 量能异常检测

**指标**：
- 成交量（volume）
- 成交额（amount）
- 换手率（turnover_rate）

**算法**：IQR（四分位距）

```
判断逻辑：
1. 计算近30天成交量的 Q1(25%) 和 Q3(75%)
2. IQR = Q3 - Q1
3. 正常范围：[Q1 - 1.5×IQR, Q3 + 1.5×IQR]
4. 超出范围 → 异常（放量或缩量）
```

**阈值**：
- 放量 > Q3 + 1.5×IQR
- 缩量 < Q1 - 1.5×IQR

### 3.4 波动异常检测

**指标**：
- 日内振幅 = (high - low) / close × 100%

**算法**：百分位法

```
判断逻辑：
1. 计算近30天振幅的分布
2. 当前振幅处于历史95%以上 → 异常（振幅过大）
3. 当前振幅处于历史5%以下 → 异常（振幅过小）
```

### 3.5 趋势异常检测

**指标**：
- MA5（5日均线）
- MA20（20日均线）
- MACD
- 布林带（BOLL）

**检测项**：

| 检测项 | 条件 | 含义 |
|--------|------|------|
| 均线死叉 | MA5 下穿 MA20 | 多头转空头 |
| 均线金叉 | MA5 上穿 MA20 | 空头转多头 |
| 跌破均线 | 收盘价 < MA20 | 趋势转弱 |
| 突破布林上轨 | 收盘价 > BOLL_upper | 可能见顶 |
| 跌破布林下轨 | 收盘价 < BOLL_lower | 可能见底 |

### 3.6 综合评分

```
异常分数 = Σ(各检测器分数 × 权重)

权重分配：
- 价格异常：30%
- 量能异常：25%
- 波动异常：20%
- 趋势异常：25%

异常等级：
- critical: score >= 0.9
- high:     score >= 0.7
- medium:   score >= 0.4
- low:      score > 0
- normal:   score = 0
```
