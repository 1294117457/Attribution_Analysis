# 技术指标异常检测指南

## 一、概述

本指南聚焦于**移动平均线（MA/EMA）**、**MACD**、**KDJ**、**RSI**、**布林带（BOLL）** 五种技术指标，及其对应的异常检测方法。

### 指标分类

| 类别 | 指标 | 核心作用 |
|------|------|----------|
| 趋势类 | MA、EMA、MACD | 判断方向、涨跌力度 |
| 超买超卖类 | KDJ、RSI | 判断是否涨/跌过头 |
| 波动类 | 布林带 | 判断价格是否离谱 |

---

## 二、移动平均线（MA/EMA）

### 2.1 什么是MA/EMA

| 指标 | 全称 | 计算方式 |
|------|------|----------|
| **MA** | 简单移动平均 | (P1+P2+...+Pn) / n，所有天数平等对待 |
| **EMA** | 指数移动平均 | 越近的天数权重越大 |

```
MA5 = 近5天收盘价的简单平均
MA20 = 近20天收盘价的简单平均（中期趋势）
EMA12 = 近12天收盘价的指数平均（反应快）

MACD使用的是EMA，不是MA
```

### 2.2 异常检测方法

#### 方法1：价格偏离均线

**原理**：价格偏离均线过远，会均值回归

```python
偏离度 = (当前价格 - MA20) / MA20 * 100

异常阈值：
- 偏离度 > +10% → 价格偏高（异常）
- 偏离度 < -10% → 价格偏低（异常）
```

#### 方法2：均线死叉/金叉检测

**原理**：MA5穿过MA20形成买卖信号

| 形态 | 检测条件 | 含义 |
|------|----------|------|
| **金叉** | MA5从下方上穿MA20 | 买入信号 |
| **死叉** | MA5从上方下穿MA20 | 卖出信号 |

```python
if 昨日MA5 < 昨日MA20 and 今日MA5 > 今日MA20:
    信号 = "金叉"
elif 昨日MA5 > 昨日MA20 and 今日MA5 < 今日MA20:
    信号 = "死叉"
```

#### 方法3：均线多头/空头排列

**原理**：多条均线有序排列确认趋势

```python
多头排列 = MA5 > MA20 > MA60
空头排列 = MA5 < MA20 < MA60

异常检测：
- 从多头排列变为空头排列 → 趋势反转（异常）
- 从空头排列变为多头排列 → 趋势反转（异常）
```

### 2.3 检测规则汇总

| 异常类型 | 检测条件 | 风险等级 |
|----------|----------|----------|
| 价格偏高 | 偏离MA20 > 10% | 中 |
| 价格偏低 | 偏离MA20 < -10% | 中 |
| 趋势反转 | 金叉/死叉发生 | 高 |
| 趋势破坏 | 多头→空头排列 | 高 |

## 三、MACD（指数平滑异同移动平均线）

### 3.1 什么是MACD

```
EMA12 = 近12天收盘价的指数平均（反应快）
EMA26 = 近26天收盘价的指数平均（反应慢）

DIF = EMA12 - EMA26        # 快线
DEA = DIF的9日EMA          # 信号线（慢线）
MACD柱 = (DIF - DEA) × 2  # 红绿柱子
```

### 3.2 核心概念

| 概念 | 含义 |
|------|------|
| **零轴** | MACD的分界线，DIF和DEA的参考点 |
| **零轴上方** | EMA12 > EMA26，整体多头 |
| **零轴下方** | EMA12 < EMA26，整体空头 |
| **金叉** | DIF上穿DEA，买入信号 |
| **死叉** | DIF下穿DEA，卖出信号 |

### 3.3 异常检测方法

#### 方法1：零轴穿越检测

```python
if 昨日MACD < 0 and 今日MACD > 0:
    信号 = "MACD上穿零轴（多头信号）"
elif 昨日MACD > 0 and 今日MACD < 0:
    信号 = "MACD下穿零轴（空头信号）"

异常：MACD从负转正或从正转负 → 趋势可能反转
```

#### 方法2：金叉/死叉检测

```python
if 昨日DIF < 昨日DEA and 今日DIF > 今日DEA:
    信号 = "MACD金叉（买入信号）"
elif 昨日DIF > 昨日DEA and 今日DIF < 今日DEA:
    信号 = "MACD死叉（卖出信号）"

异常：高位死叉（MACD>0区域死叉）→ 可能是假信号
```

#### 方法3：顶背离/底背离

**原理**：价格创新高/低，但MACD没有跟上 → 可能反转

```python
# 顶背离（看跌信号）
if 价格创N日新高 and DIF没有创新高:
    信号 = "顶背离（可能下跌）"

# 底背离（看涨信号）
if 价格创N日新低 and DIF没有创新低:
    信号 = "底背离（可能上涨）"
```

#### 方法4：MACD柱形态异常

```python
# 柱子由绿转红（动能转强）
if 昨日MACD柱 < 0 and 今日MACD柱 > 0:
    信号 = "绿柱转红柱（多头动能增强）"

# 柱子持续缩短（动能减弱）
if 连续N日MACD柱都在缩短 and 价格还在涨:
    信号 = "顶背离前兆（动能衰竭）"
```

### 3.4 检测规则汇总

| 异常类型 | 检测条件 | 风险等级 |
|----------|----------|----------|
| 零轴穿越 | DIF穿过零轴 | 高 |
| 金叉/死叉 | DIF穿过DEA | 中 |
| 顶背离 | 价格新高但DIF没有 | 高 |
| 底背离 | 价格新低但DIF没有 | 高 |
| 动能衰竭 | MACD柱持续缩短 | 中 |

## 四、KDJ（随机指标）

### 4.1 什么是KDJ

```
RSV = (今日收盘价 - N日最低价) / (N日最高价 - N日最低价) × 100

K = 2/3 × 昨日K + 1/3 × RSV
D = 2/3 × 昨日D + 1/3 × K
J = 3 × K - 2 × D
```

### 4.2 取值范围与含义

```
KDJ取值：0 ~ 100

100 ─────────────────────────────────
    │     超买区
80  ├─────────────────────────────┤ ← 考虑卖出
    │     正常区间
20  ├─────────────────────────────┤ ← 考虑买入
    │     超卖区
 0 ─────────────────────────────────
```

| 值 | 区域 | 含义 |
|----|------|------|
| > 80 | 超买区 | 涨过头，可能要跌 |
| < 20 | 超卖区 | 跌过头，可能要涨 |

### 4.3 异常检测方法

#### 方法1：超买/超卖检测

```python
if KDJ > 80:
    信号 = "超买（可能回调）"
elif KDJ < 20:
    信号 = "超卖（可能反弹）"
```

#### 方法2：KDJ金叉/死叉

```python
if 昨日K < 昨日D and 今日K > 今日D:
    信号 = "KDJ金叉（买入信号）"
elif 昨日K > 昨日D and 今日K < 今日D:
    信号 = "KDJ死叉（卖出信号）"

异常：高位金叉（K>80时金叉）→ 可能是假信号
异常：低位死叉（K<20时死叉）→ 可能是假信号
```

#### 方法3：KDJ背离

```python
# 顶背离（看跌）
if 价格创N日新高 and KDJ没有创新高:
    信号 = "KDJ顶背离"

# 底背离（看涨）
if 价格创N日新低 and KDJ没有创新低:
    信号 = "KDJ底背离"
```

### 4.4 检测规则汇总

| 异常类型 | 检测条件 | 风险等级 |
|----------|----------|----------|
| 超买 | KDJ > 80 | 中 |
| 超卖 | KDJ < 20 | 中 |
| 高位金叉 | K>80时金叉 | 高（假信号概率大） |
| 低位死叉 | K<20时死叉 | 高（假信号概率大） |
| 顶背离 | 价格新高但KDJ没有 | 高 |
| 底背离 | 价格新低但KDJ没有 | 高 |

## 五、RSI（相对强弱指数）

### 5.1 什么是RSI

```
上涨幅度 = 今日收盘价 - 昨日收盘价（若为正）
下跌幅度 = 昨日收盘价 - 今日收盘价（若为负）

RS = N天上涨幅度均值 / N天下跌幅度均值
RSI = 100 - (100 / (1 + RS))
```

### 5.2 取值范围与含义

```
RSI取值：0 ~ 100

100 ─────────────────────────────────
    │     超买区
70  ├─────────────────────────────┤ ← 可能要跌
    │     正常区间 (30~70)
30  ├─────────────────────────────┤ ← 可能要涨
    │     超卖区
 0 ─────────────────────────────────
```

| RSI值 | 含义 |
|-------|------|
| > 70 | 超买，涨过头了 |
| < 30 | 超卖，跌过头了 |
| = 50 | 多空平衡 |

### 5.3 异常检测方法

#### 方法1：超买/超卖检测

```python
if RSI > 70:
    信号 = "超买（可能回调）"
elif RSI < 30:
    信号 = "超卖（可能反弹）"

异常阈值：
- RSI > 80 → 严重超买
- RSI < 20 → 严重超卖
```

#### 方法2：RSI背离

```python
# 顶背离（看跌）
if 价格创N日新高 and RSI没有创新高:
    信号 = "RSI顶背离"

# 底背离（看涨）
if 价格创N日新低 and RSI没有创新低:
    信号 = "RSI底背离"
```

#### 方法3：RSI钝化

**原理**：RSI长期处于高位/低位，可能失去信号意义

```python
# RSI持续高位
if 连续N天RSI > 70:
    信号 = "RSI高位钝化（趋势强劲但警惕反转）"

# RSI持续低位
if 连续N天RSI < 30:
    信号 = "RSI低位钝化（趋势弱势但警惕反弹）"
```

#### 方法4：RSI与价格背离 + 超买超卖组合

```python
# 最强卖出信号
if 价格创近期新高 and RSI创近期新高 and RSI > 70:
    信号 = "高位共振（双重确认）"

# 最强买入信号
if 价格创近期新低 and RSI创近期新低 and RSI < 30:
    信号 = "低位共振（双重确认）"
```

### 5.4 检测规则汇总

| 异常类型 | 检测条件 | 风险等级 |
|----------|----------|----------|
| 超买 | RSI > 70 | 中 |
| 超卖 | RSI < 30 | 中 |
| 严重超买 | RSI > 80 | 高 |
| 严重超卖 | RSI < 20 | 高 |
| 顶背离 | 价格新高但RSI没有 | 高 |
| 底背离 | 价格新低但RSI没有 | 高 |
| 高位钝化 | RSI持续 > 70 | 中 |

## 六、布林带（BOLL）

### 6.1 什么是布林带

```
中轨 = MA20（20日均线）
标准差 = 价格与MA20偏差的统计值

上轨 = MA20 + 2 × 标准差
下轨 = MA20 - 2 × 标准差
```

### 6.2 图示

```
            上轨（偏高）
              │
    ──────────┼───────────
              │ 中轨(MA20)
              │
    ──────────┼───────────
            下轨（偏低）

价格通常在上下轨之间波动
统计学上：约95%的价格落在此区间内
```

### 6.3 异常检测方法

#### 方法1：价格突破布林带

```python
if 价格 > 上轨:
    信号 = "突破上轨（价格偏高，可能回调）"
elif 价格 < 下轨:
    信号 = "突破下轨（价格偏低，可能反弹）"

异常程度：
- 偏离上轨 5% 以内 → 轻度超买
- 偏离上轨 > 5% → 重度超买
- 偏离下轨 5% 以内 → 轻度超卖
- 偏离下轨 > 5% → 重度超卖
```

#### 方法2：布林带开口/收口

```python
# 布林带收口（波动减小）
if (上轨 - 下轨) 持续缩小:
    信号 = "布林带收口（波动减小，可能大幅波动）"

# 布林带开口（波动加大）
if (上轨 - 下轨) 持续扩大:
    信号 = "布林带开口（波动加大）"

# 异常：开口后价格突破
if 布林带开口 and (价格突破上轨 or 价格跌破下轨):
    信号 = "开口突破（趋势信号强）"
```

#### 方法3：布林带与RSI共振

```python
# 双重超买信号
if 价格 > 上轨 and RSI > 70:
    信号 = "双重超买（价格+RSI都偏高）"

# 双重超卖信号
if 价格 < 下轨 and RSI < 30:
    信号 = "双重超卖（价格+RSI都偏低）"
```

#### 方法4：价格贴轨/沿着轨道运行

```python
# 价格持续贴近上轨
if 连续N天 价格 > 上轨 × 0.98:
    信号 = "价格紧贴上轨（强势上涨）"

# 价格持续贴近下轨
if 连续N天 价格 < 下轨 × 1.02:
    信号 = "价格紧贴下轨（弱势下跌）"
```

### 6.4 检测规则汇总

| 异常类型 | 检测条件 | 风险等级 |
|----------|----------|----------|
| 突破上轨 | 价格 > 上轨 | 高 |
| 突破下轨 | 价格 < 下轨 | 高 |
| 开口突破 | 开口 + 突破轨道 | 高 |
| 收口蓄势 | 布林带持续收口 | 中（可能大波动） |
| 双重共振 | 布林带 + RSI | 高 |

## 七、综合异常检测框架

### 7.1 检测流程

```
实时数据
    │
    ├── 价格数据
    │     ↓
    ├── 计算MA/EMA/MACD/KDJ/RSI/BOLL
    │     ↓
    ├── 逐指标检测异常
    │     ↓
    └── 综合判断 → 输出异常报告
```

### 7.2 异常等级定义

| 等级 | 说明 | 处理方式 |
|------|------|----------|
| **低风险** | 单指标轻微异常 | 关注 |
| **中风险** | 单指标明显异常 / 多指标轻微异常 | 预警 |
| **高风险** | 多指标明显异常 / 背离信号 | 强烈预警 |

### 7.3 组合信号矩阵

| 信号类型 | 包含指标 | 风险等级 |
|----------|----------|----------|
| **超买共振** | RSI>70 + KDJ>80 + 价格>布林上轨 | 高 |
| **超卖共振** | RSI<30 + KDJ<20 + 价格<布林下轨 | 高 |
| **顶背离共振** | 价格新高 + MACD顶背离 + RSI顶背离 | 高 |
| **底背离共振** | 价格新低 + MACD底背离 + RSI底背离 | 高 |
| **趋势破坏** | 死叉 + MACD下零轴 + 空头排列 | 高 |

### 7.4 输出格式示例

```python
异常报告 = {
    "timestamp": "2024-01-15 10:30:00",
    "stock_code": "600519",
    "异常等级": "高",
    "异常类型": "超买共振",
    "指标详情": {
        "RSI": 78.5,      # 超买
        "KDJ": 85.2,      # 超买
        "布林带": "突破上轨",
        "MACD": "零轴上方",
        "MA": "多头排列"
    },
    "建议": "考虑减仓，警惕回调风险"
}
```

## 八、Python实现示例

### 8.1 基础指标计算

```python
import pandas as pd
import numpy as np

def calculate_ma(close, period):
    """计算MA"""
    return close.rolling(window=period).mean()

def calculate_ema(close, period):
    """计算EMA"""
    return close.ewm(span=period, adjust=False).mean()

def calculate_macd(close, fast=12, slow=26, signal=9):
    """计算MACD"""
    ema_fast = calculate_ema(close, fast)
    ema_slow = calculate_ema(close, slow)
    dif = ema_fast - ema_slow
    dea = calculate_ema(dif, signal)
    macd_hist = (dif - dea) * 2
    return dif, dea, macd_hist

def calculate_kdj(high, low, close, period=9):
    """计算KDJ"""
    low_n = low.rolling(window=period).min()
    high_n = high.rolling(window=period).max()
    rsv = (close - low_n) / (high_n - low_n) * 100
    k = rsv.ewm(alpha=1/3, adjust=False).mean()
    d = k.ewm(alpha=1/3, adjust=False).mean()
    j = 3 * k - 2 * d
    return k, d, j

def calculate_rsi(close, period=14):
    """计算RSI"""
    delta = close.diff()
    gain = delta.where(delta > 0, 0)
    loss = -delta.where(delta < 0, 0)
    avg_gain = gain.ewm(alpha=1/period, adjust=False).mean()
    avg_loss = loss.ewm(alpha=1/period, adjust=False).mean()
    rs = avg_gain / avg_loss
    rsi = 100 - (100 / (1 + rs))
    return rsi

def calculate_boll(close, period=20, std_dev=2):
    """计算布林带"""
    ma = calculate_ma(close, period)
    std = close.rolling(window=period).std()
    upper = ma + std_dev * std
    lower = ma - std_dev * std
    return upper, ma, lower
```

### 8.2 异常检测示例

```python
def detect_anomalies(df):
    """综合异常检测"""
    results = []

    # 计算指标
    df['MA5'] = calculate_ma(df['close'], 5)
    df['MA20'] = calculate_ma(df['close'], 20)
    df['dif'], df['dea'], df['macd'] = calculate_macd(df['close'])
    df['k'], df['d'], df['j'] = calculate_kdj(df['high'], df['low'], df['close'])
    df['rsi'] = calculate_rsi(df['close'])
    df['upper'], df['middle'], df['lower'] = calculate_boll(df['close'])

    # 检测异常
    anomalies = []

    # RSI超买/超卖
    if df['rsi'].iloc[-1] > 70:
        anomalies.append({"type": "RSI超买", "level": "中", "value": df['rsi'].iloc[-1]})
    elif df['rsi'].iloc[-1] < 30:
        anomalies.append({"type": "RSI超卖", "level": "中", "value": df['rsi'].iloc[-1]})

    # KDJ超买/超卖
    if df['j'].iloc[-1] > 80:
        anomalies.append({"type": "KDJ超买", "level": "中", "value": df['j'].iloc[-1]})
    elif df['j'].iloc[-1] < 20:
        anomalies.append({"type": "KDJ超卖", "level": "中", "value": df['j'].iloc[-1]})

    # 布林带突破
    if df['close'].iloc[-1] > df['upper'].iloc[-1]:
        anomalies.append({"type": "突破布林带上轨", "level": "高", "value": df['close'].iloc[-1]})
    elif df['close'].iloc[-1] < df['lower'].iloc[-1]:
        anomalies.append({"type": "突破布林带下轨", "level": "高", "value": df['close'].iloc[-1]})

    # MACD零轴
    if df['dif'].iloc[-1] > 0 and df['dif'].iloc[-2] <= 0:
        anomalies.append({"type": "MACD上穿零轴", "level": "高"})
    elif df['dif'].iloc[-1] < 0 and df['dif'].iloc[-2] >= 0:
        anomalies.append({"type": "MACD下穿零轴", "level": "高"})

    return anomalies
```

## 九、参考阈值汇总

| 指标 | 异常阈值 | 异常类型 |
|------|----------|----------|
| **MA偏离度** | > 10% 或 < -10% | 价格偏高/偏低 |
| **MACD** | 穿越零轴 | 趋势确认 |
| **MACD背离** | 价格创新高/低但DIF没有 | 可能反转 |
