# 技术指标与异常检测

## 概述

本文档介绍 MA、EMA、MACD、RSI、KDJ、BOLL 六个常用技术指标的含义、作用、异常检测方法。

---

## 1. MA（移动平均线）

### 含义
一段时间内的收盘价平均值。

```
MA5  = 最近5天收盘价的平均
MA20 = 最近20天收盘价的平均
MA60 = 最近60天收盘价的平均
```

### 作用
- **趋势判断**：MA 向上 → 上涨趋势；MA 向下 → 下跌趋势
- **支撑阻力**：价格在 MA 附近获得支撑或压力
- **金叉死叉**：短期 MA 上穿/下穿长期 MA

### 使用方法
| 信号 | 条件 | 含义 |
|------|-------|------|
| **多头排列** | MA5 > MA20 > MA60 | 强势上涨 |
| **空头排列** | MA5 < MA20 < MA60 | 强势下跌 |
| **金叉** | MA5 上穿 MA20 | 看涨信号 |
| **死叉** | MA5 下穿 MA20 | 看跌信号 |

### 异常检测
```python
# 均线偏离度异常
deviation = (price - MA20) / MA20
if abs(deviation) > 0.10:  # 偏离超过10%
    异常 = True
```

## 2. EMA（指数移动平均线）

### 含义
对近期数据赋予更大权重的移动平均线，比 MA 更敏感。

```
EMA_today = (收盘价 × k) + (昨日EMA × (1-k))
k = 2 / (周期 + 1)
```

### 与 MA 的区别
| 特性 | MA | EMA |
|------|-----|-----|
| 灵敏度 | 低（平均对待） | 高（重视近期） |
| 稳定性 | 好 | 差 |
| 适用 | 长期趋势 | 短期交易 |

### 作用
- 更灵敏地反映价格变化
- 常用于 MACD、RSI 等指标的计算基础

### 异常检测
```python
# EMA 交叉异常（快速变化）
if EMA5 突然上穿 EMA20 且幅度 > 5%:
    异常 = True
```

## 3. MACD（移动平均收敛发散）

### 含义
由快线(DIF)、慢线(DEA)、柱状图(MACD柱) 组成。

```
DIF  = EMA12 - EMA26        # 快线
DEA  = DIF的EMA9            # 慢线
MACD柱 = (DIF - DEA) × 2    # 柱状图
```

### 作用
- **趋势判断**：DIF > DEA → 多头；DIF < DEA → 空头
- **动能判断**：MACD 柱放大 → 动能增强
- **背离判断**：价格创新高但 MACD 没新高 → 顶背离

### 使用方法
| 信号 | 条件 | 含义 |
|------|-------|------|
| **金叉** | DIF 上穿 DEA | 看涨 |
| **死叉** | DIF 下穿 DEA | 看跌 |
| **零轴上方** | DIF > 0 | 上涨动能 |
| **零轴下方** | DIF < 0 | 下跌动能 |
| **顶背离** | 价格↑ 但 DIF↓ | 可能见顶 |
| **底背离** | 价格↓ 但 DIF↑ | 可能见底 |

### 异常检测
```python
# MACD 柱异常放大
if MACD柱 > 历史MACD柱.max() × 1.5:
    异常 = True

# DIF 穿越零轴
if DIF 刚从负转正 or 从正转负:
    异常 = True

# 背离检测
if 价格创新高 and DIF < 前高:
    异常 = "顶背离"
```

## 4. RSI（相对强弱指数）

### 含义
衡量价格涨跌动能的指标，取值 0-100。

```
RS = 平均涨幅 / 平均跌幅
RSI = 100 - (100 / (1 + RS))
```

### 作用
- **超买超卖**：判断价格是否过热或过冷
- **趋势确认**：RSI 与价格走势配合

### 使用方法
| 区间 | RSI 值 | 含义 |
|------|--------|------|
| 超买区 | RSI > 70 | 可能回调 |
| 中性区 | 30 ≤ RSI ≤ 70 | 正常波动 |
| 超卖区 | RSI < 30 | 可能反弹 |

### 异常检测
```python
# RSI 超买超卖
if RSI > 80:
    异常类型 = "严重超买"
elif RSI > 70:
    异常类型 = "超买"
elif RSI < 20:
    异常类型 = "严重超卖"
elif RSI < 30:
    异常类型 = "超卖"

# RSI 背离
if 价格创新低 and RSI > 前低RSI:
    异常 = "底背离"
```

## 5. KDJ（随机指标）

### 含义
随机指标，由 K 线、D 线、J 线组成，取值 0-100。

```
RSV = (收盘价 - N日最低) / (N日最高 - N日最低) × 100
K = SMA(RSV, 3)
D = SMA(K, 3)
J = 3K - 2D
```

### 作用
- **超买超卖**：类似 RSI
- **交叉信号**：K 线与 D 线的交叉
- **钝化识别**：强势趋势中 KDJ 可能持续高位/低位

### 使用方法
| 区间 | KDJ 值 | 含义 |
|------|--------|------|
| 超买区 | K/D/J > 80 | 可能回调 |
| 中性区 | 20 ≤ K/D ≤ 80 | 正常 |
| 超卖区 | K/D/J < 20 | 可能反弹 |

| 信号 | 条件 | 含义 |
|------|-------|------|
| **金叉** | K 上穿 D | 看涨 |
| **死叉** | K 下穿 D | 看跌 |
| **J > 100** | J 值过高 | 超买预警 |
| **J < 0** | J 值过低 | 超卖预警 |

### 异常检测
```python
# KDJ 极端值
if K > 90 or D > 90 or J > 100:
    异常 = "KDJ 严重超买"
if K < 10 or D < 10 or J < 0:
    异常 = "KDJ 严重超卖"

# KDJ 钝化（持续高位/低位）
if K > 80 持续超过5天:
    异常 = "KDJ 高位钝化（可能继续涨）"
```

## 6. BOLL（布林带）

### 含义
由中轨、上轨、下轨组成，反映价格波动范围。

```
中轨 = MA20
上轨 = MA20 + 2 × 标准差
下轨 = MA20 - 2 × 标准差
```

### 作用
- **波动率指标**：带宽大 → 波动大；带宽小 → 波动小
- **支撑阻力**：上轨/下轨作为动态支撑/压力
- **突破信号**：价格突破布林带

### 使用方法
| 位置 | 含义 |
|------|------|
| **触及上轨** | 价格强势，可能回调 |
| **触及下轨** | 价格弱势，可能反弹 |
| **突破上轨** | 极度强势（注意假突破） |
| **突破下轨** | 极度弱势（注意假突破） |
| **收窄开口** | 酝酿大行情 |

### 异常检测
```python
# 轨道突破
if price > 上轨:
    异常 = "突破上轨（极度强势）"
elif price < 下轨:
    异常 = "突破下轨（极度弱势）"

# 带宽异常（波动率变化）
bandwidth = (上轨 - 下轨) / 中轨
if bandwidth < 历史bandwidth的10%分位:
    异常 = "布林带收窄（酝酿突破）"

# 价格贴近轨道
if price > 上轨 × 0.98:
    异常 = "逼近上轨"
```

## 异常检测汇总

### 检测规则表

| 指标 | 异常条件 | 异常类型 |
|------|----------|----------|
| **MA** | 价格偏离 MA20 > 10% | 均线偏离 |
| **EMA** | 短期 EMA 快速穿越长期 EMA | 趋势突变 |
| **MACD** | MACD 柱创历史新高/新低 | 动能异常 |
| **MACD** | 价格与 DIF 背离 | 顶/底背离 |
| **RSI** | RSI > 70 或 RSI < 30 | 超买/超卖 |
| **RSI** | RSI > 80 或 RSI < 20 | 严重超买/超卖 |
| **KDJ** | J > 100 或 J < 0 | 极端值 |
| **KDJ** | K/D 持续 > 80 或 < 20 | 钝化 |
| **BOLL** | 价格突破上轨/下轨 | 轨道突破 |
| **BOLL** | 带宽极度收窄 | 酝酿突破 |

### 组合异常检测

```python
# 多指标共振异常
signals = []

if RSI > 70:
    signals.append("RSI超买")
if K > 80:
    signals.append("KDJ超买")
if price > BOLL上轨:
    signals.append("BOLL突破")
if DIF < 0:
    signals.append("MACD空头")

# 3个以上指标同时异常 = 强信号
if len(signals) >= 3:
    异常置信度 = "高"
```
