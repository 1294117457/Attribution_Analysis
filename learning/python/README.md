# Python 基础学习路径

## 学习目标
掌握 Python 基础语法，为后续 FastAPI 和 LangGraph 学习打基础。

## 学习顺序

### 第一阶段：基础语法（Day 1-2）

| Step | 标题 | 内容 | 预计时间 |
|------|------|------|----------|
| 01 | 变量和数据类型 | 字符串、数字、布尔值 | 30分钟 |
| 02 | 条件判断和循环 | if/else, for, while | 40分钟 |
| 03 | 列表和字典 | 容器类型操作 | 40分钟 |

### 第二阶段：函数和模块（Day 3-4）

| Step | 标题 | 内容 | 预计时间 |
|------|------|------|----------|
| 04 | 函数定义 | def, 参数, 返回值 | 40分钟 |
| 05 | 模块和导入 | import, from...import | 30分钟 |

### 第三阶段：面向对象（Day 5-6）

| Step | 标题 | 内容 | 预计时间 |
|------|------|------|----------|
| 06 | 类和对象 | class, self, \_\_init\_\_ | 50分钟 |
| 07 | 继承和多态 | extends, override | 40分钟 |

### 第四阶段：实战技能（Day 7）

| Step | 标题 | 内容 | 预计时间 |
|------|------|------|----------|
| 08 | 异常处理 | try/except, raise | 30分钟 |
| 09 | 文件操作 | 读写文件, with 语句 | 40分钟 |

## 学习方法
1. 先阅读 `docs/step/` 中的指导
2. 自己动手敲代码，不要复制粘贴
3. 对照 `docs/analysis/` 理解原理
4. 完成练习后运行验证

## 运行环境
```bash
cd learning/python
source ../venv/bin/activate  # 激活虚拟环境
python demo_01.py            # 运行你的代码
```
