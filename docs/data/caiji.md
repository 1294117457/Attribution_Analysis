## 数据采集全景图（按阶段）

### 第一阶段：入门必备（能跑起来）

┌─────────────────────────────────────────────────────────────┐

│  数据类型          │  用途                    │  获取难度  │

├─────────────────────────────────────────────────────────────┤

│  📊 日K线数据      │  一切分析的基础           │  ⭐ AkShare直接获取  │

│  （OHLC+成交量）   │  异常检测、画图、技术指标  │            │

│                                                             │

│  ✅ 推荐：stock_zh_a_hist() 获取所有A股日K线                │

└─────────────────────────────────────────────────────────────┘

为什么先要日K线？

- 异常检测：股价今天涨了5%，是异常吗？→ 和历史K线比
- 归因分析：涨的原因是放量？→ 看成交量变化
- 技术指标：MACD/KDJ → 都从K线算出来

┌─────────────────────────────────────────────────────────────┐

│  技术指标          │  异常检测场景              │  获取方式  │

│  📊 MACD           │  金叉/死叉、背离            │  AkShare  │

│                                                             │

│  📉 KDJ            │  超买(>80)/超卖(<20)       │  AkShare  │

│                    │  持续超买可能回调          │            │

│                                                             │

│  📐 布林带 BOLL     │  股价突破布林带上轨=异常    │  AkShare  │

│                    │  跌破下轨=超卖            │            │

│                                                             │

这些数据让你能回答：

- "茅台今天跌了3%，算异常吗？" → 和MA20比，在均线下方就是弱
- "为什么放量下跌？" → 资金在出逃

### 第三阶段：归因分析需要的上下文

┌─────────────────────────────────────────────────────────────┐

│  上下文数据        │  归因分析价值              │  获取难度  │

├─────────────────────────────────────────────────────────────┤

│  🏢 行业板块        │  是个股风险还是板块联动？   │  AkShare  │

│                    │  白酒板块集体下跌 vs 茅台自身 │            │

│                                                             │

│  📰 公告/新闻       │  有利好/利空公告吗？        │  需额外源  │

│                    │  业绩暴雷、高管减持等       │            │

│                                                             │

│  💰 资金流向        │  主力在买入还是卖出？       │  AkShare  │

│  （主力/散户）     │  主力净流出=机构不看好      │            │

│                                                             │

│  📊 大盘指数        │  是系统性风险还是个股？     │  AkShare  │

这些数据让你能回答：

- "为什么茅台跌了？" → "因为白酒板块今天整体跌了2%"
