# Backend 功能文档

## 概述

Backend 是智能金融数据归因分析平台的后端服务，基于 **FastAPI + Python** 构建，负责数据采集、异常检测、归因分析等核心功能。

**当前阶段**：Phase 1-2（数据采集 + 数据库）

---

## 功能模块

### 1. 数据采集模块 (`data/`)

| 功能 | 状态 | 说明 |
|------|------|------|
| AkShare 采集 | ✅ | 获取 A 股 K 线数据 |
| 数据解析 | ✅ | DataFrame → StockKline |
| 批量采集 | ✅ | 多只股票同时采集 |
| 数据存储 | ✅ | 存入 PostgreSQL |
| 命令行脚本 | ✅ | `python -m data.scripts.collect_stock` |

**数据源**：AkShare 1.14（免费 A 股接口）

**采集数据类型**：
- 日 K 线（OHLCV）
- 实时行情

### 2. API 网关 (`app/api/`)

| 接口 | 方法 | 说明 |
|------|------|------|
| `GET /api/v1/health` | GET | 健康检查 |
| `GET /api/v1/stocks/` | GET | 查询 K 线 |
| `GET /api/v1/stocks/{symbol}` | GET | 获取指定股票 |
| `POST /api/v1/stocks/collect` | POST | 采集并存储 |

## 使用方式

### 启动服务

```bash
cd backend
uvicorn app.main:app --reload
```

### 命令行采集

```bash
# 单只股票
python -m data.scripts.collect_stock 000001

# 批量采集
python -m data.scripts.batch_collect
```

### API 调用

```bash
# 健康检查
curl http://localhost:8000/api/v1/health

# 查询 K 线
curl "http://localhost:8000/api/v1/stocks/?symbol=000001&limit=10"

# 采集数据
curl -X POST "http://localhost:8000/api/v1/stocks/collect" \
  -H "Content-Type: application/json" \
  -d '{"symbol": "000001", "days": 30}'
```

## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| Web 框架 | FastAPI 0.115 | 异步高性能 |
| 数据库 | PostgreSQL | 关系型 + pgvector |
| ORM | SQLAlchemy 2.0 | 类型安全 |
| 数据采集 | AkShare | 免费 A 股数据 |
| 缓存 | Redis | 会话/热点数据 |
