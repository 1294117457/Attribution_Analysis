## 核心特性

- **多 Agent 协作**：LangGraph 编排异常检测 → 情感分析 → 归因分析 → 报告生成
- **双引擎架构**：算法保证精确 + AI 理解业务上下文
- **pgvector 经验累积**：PostgreSQL 内嵌向量，相似案例匹配，系统越用越聪明
- **Redis 高性能缓存**：热点数据缓存、API 限流
- **Docker 微服务**：基于 Docker Compose 的微服务模式
- **金融垂直场景**：专注 A 股市场数据分析
