## 核心特性

- **多 Agent 协作**：LangGraph 编排异常检测 → 归因分析 → 报告生成
- **双引擎架构**：算法保证精确 + AI 理解业务上下文
- **经验累积机制**：向量检索匹配历史经验，系统越用越聪明
- **金融垂直场景**：专注利率、汇率、加密货币等专业数据
